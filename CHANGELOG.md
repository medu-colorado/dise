# Changelog

All notable changes to this project will be documented in this file.
Format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.1.0] — 2025-06-10
### Added
- `DirectionalSensitivityExplainer` — core perturbation engine for classification and regression
- Four distance methods: percentile, Wasserstein, Mahalanobis, minimal counterfactual
- Robust continuous/discrete feature detector with object-array and dummy-encoding support
- `ExplainerPlotter` for visualising threshold-crossing events
- Full test suite (50 tests)
